WHAT IS EXPLORATOR?

Explorator is a 2D visual game where players will have to find the shortest and efficient path from node A to node B. Players will be able to use their mouse in order to create a path to arrive at the finishing node (Node B). Once they have created their path, Dijkstra's algorithm will run in the background in order to find the shortest path. There will be a score betweeen 0-100% to determine how close the player was to creating the shortest path possible.

The starting node (Node A) is highlighted in yellow and the ending node (Node B) is highlighted in green. To start creating a path, click over the starting node(yellow) and drag the mouse over the transparent nodes while still having the mouse clicked. The black nodes are walls which you are not able to pass through, and if the mouse has entered the wall, you will need to click on the last node you have left to keep creating the path. You can keep creating a path until you have reached the finishing node(Green).

Once you have finished creating your path, you can now display dijkstra's algorithm which at the end, will display the score to you. To look at the solution, press the solution button after clicking the display button. The new path is the actual shortest path possible here and if there is any green nodes, then that means you are on the correct path.
