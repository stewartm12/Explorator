WHAT IS EXPLORATOR?

Explorator is a 2D visual game where players will have to find the shortest path from the white node to the green node.

Players will be able to create a path by clicking and dragging towards the green node.

Once they have created their path, Dijkstra's algorithm will run in the background in order to find the shortest path.

There will be a score betweeen 0-100% to determine how close the player was to creating the shortest path.
